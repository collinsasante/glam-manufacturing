# GlamPack Warehouse Management System

A professional, modern warehouse and supply chain management web application.

## Quick Start

```bash
npm run dev
```

Open http://localhost:3000

## Features

- Dashboard with real-time statistics
- Raw Materials inventory management
- Stock Movement tracking
- Stock Transfer between warehouses
- Supplier management
- Professional modern UI

## Tech Stack

- Next.js 16.1 + TypeScript
- Tailwind CSS
- Airtable Backend
